# 简介
## 运行
```
开发
npm i
npm run dev
发布
npm run build
```
## 项目概述
轻量化中后台应用的开发，对于无特殊交互的中后台应用，可直接通过部署包配置json文件即可运行。
- 应用名称、菜单基于config/app.json配置；
- 页面基于内置的百度amis页面设计器配置保存为config/pages/xx.json；
## 自定义页面
- 可通过源码开发新路由，在菜单中path进行指定；
- 可通过amis的iframe引入；

## 技术栈
    react，集成了百度amis页面设计器，框架使用了`@umijs/max` 模板项目。
